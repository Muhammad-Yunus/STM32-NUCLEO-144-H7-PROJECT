## Description

![](doc/thumb-lvgl.jpeg)
This repository contains an example project for the STM32 Nucleo-144 H7 microcontroller board, specifically targeting boards like the STM32 Nucleo H753ZI. The project demonstrates best practices using:

- **STM32CubeIDE** - Integrated development environment
- **RTOS** - Real-time operating system implementation
- **Multimedia** - Audio/video processing capabilities
- **Edge AI** - On-device machine learning inference

This is a reference implementation suitable for developers learning embedded systems development on the STM32 H7 platform.
